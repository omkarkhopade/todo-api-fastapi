#  Todo List API (FastAPI)

A fully-featured REST API built using **FastAPI** that supports user authentication, role-based access control (Admin/User), task management, and email notifications.

---

##  Features

-  JWT Authentication (Login & Secure Access)
-  Role-Based Access (Admin & User)
-  Email Notifications (Task Assignment & Completion)
-  Task Management (Create, Update, Delete, Complete)
-  Email Verification System
-  Notification Preferences (Enable/Disable Emails)
-  Database Integration using SQLAlchemy
-  Async Email Sending using SMTP

---

##  Tech Stack

- **Backend Framework:** FastAPI
- **Database:** SQLite / PostgreSQL (configurable)
- **ORM:** SQLAlchemy
- **Authentication:** JWT (JSON Web Token)
- **Email Service:** SMTP (Gmail)
- **Templating:** Jinja2
- **Server:** Uvicorn
